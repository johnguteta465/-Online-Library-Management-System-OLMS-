
##  Online Library Management System (OLMS)

The **Online Library Management System (OLMS)** is a web-based application designed to manage library operations efficiently. It allows administrators and users to organize, access, and manage books in a digital environment.

This project was developed to demonstrate practical skills in **web development, database design, and system management** using PHP and MySQL.


##  Project Objective

The main goal of this system is to simplify library management by providing a centralized platform where books, members, and borrowing activities can be easily handled.

It helps reduce manual work and improves record accuracy in library operations.


##  Key Features

* Book Management – Add, update, delete, and view books
* ember Management – Register and manage library members
* Borrowing System – Track borrowed and returned books
* Database Integration– Store and manage data using MySQL
* Web Interface– User-friendly interface built with HTML, CSS, and PHP

---

Technologies Used

* Frontend: HTML, CSS
* Backend:PHP
* Database: MySQL
* Server Environment: XAMPP (Local Development)
* Deployment: InfinityFree


 Learning Purpose

This project was developed as part of a **database and web development learning project**. It helped strengthen my understanding of:

* Database design and relationships
* CRUD operations (Create, Read, Update, Delete)
* Backend development with PHP
* Connecting web applications with databases
* Deploying a web application to a live server
